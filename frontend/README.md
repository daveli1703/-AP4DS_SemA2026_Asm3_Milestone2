# GLŌW Frontend

React + Vite frontend for the beauty & cosmetics shopping website.

## Requirements

- Node.js 18+
- Backend running at `http://localhost:8000`

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Opens at **http://localhost:3000**

All `/api/*` requests are proxied to `http://localhost:8000` — no CORS config needed.
